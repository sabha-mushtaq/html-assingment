  1. < >[OPENING TAG]: THIS TAG IS INITIAL PART OF HTML ELEMENT.IT IS USED TO BEGIN A HTML ELEMENT
  2. < / >[CLOSING TAG]: THIS TAG REPRESENTS END PART OF A HTML ELEMENT WHICH IS USED TO TERMINATE A HTML ELEMENT.
  3. <br>[break tag]:t This tag is used to break line.It has only a opening section and no closing one.
